# cavu-dep-board

## Setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Run unit tests with coverage
```
npm run test:coverage
```

### Lints and fixes files
```
npm run lint
```
